# Calamore - Web - Client

Express server and react code for the webapp

```
├── components    # Dehydrated components
├── containers    # Hydrated components 
├── entrypoints   # Entrypoints for webpack client bundles
├── forms         # Redux forms
├── fragments     # Graphql fragments
├── hocs          # Higher order components
├── lib           # Core client side classes/objects
├── mutations     # Graphql mutations
├── pages         # Web pages (includes layouts)
├── prop_types    # React prop types
├── queries       # Graphql queries
├── redux         # Redux config/reducers/actions
└── styles        # Global css
```
